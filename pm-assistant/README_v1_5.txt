PM Tracking System v1.5 - Productivity Edition

ฐานข้อมูล:
- ใช้ pm_tracking.db เดิมจากไฟล์ v1.4.1 ที่ผู้ใช้ส่งมา
- เพิ่มเฉพาะตาราง pm_task_state สำหรับ Pause / Follow-up / Quick Note
- ไม่ล้างแผนงาน ศูนย์บริการ การตั้งค่า LINE หรือ Log เดิม

ฟีเจอร์ใหม่:
1. My Today เป็นหน้าแรก
2. Smart Priority และคิวงานประจำวัน
3. One Click Complete พร้อม PM_History
4. Pause / Resume และ Quick Follow-up
5. Morning Brief
6. Follow-up เฉพาะงานค้างที่ไม่ได้ Pause
7. Evening Summary
8. Weekly Summary
9. Inbox Zero / Today's Mission Complete
10. Responsive layout สำหรับหน้าจอเล็ก

การเริ่มใช้งาน:
1. แตก ZIP
2. เปิด run.bat
3. หน้าแรกคือ My Today
4. เปิดสวิตช์แจ้งเตือนลดภาระงานเมื่อต้องการใช้งาน Scheduler

หมายเหตุ:
- การแจ้งเตือนจากเครื่องส่วนตัวต้องเปิดโปรแกรมไว้
- Webhook จาก LINE ต้องเปิด ngrok ไว้ตามเดิม
- ควรสำรอง pm_tracking.db ก่อนอัปเดตเวอร์ชันทุกครั้ง
