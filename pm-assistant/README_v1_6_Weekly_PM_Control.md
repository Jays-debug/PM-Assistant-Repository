# PM Assistant v1.6 – Weekly PM Control

## เป้าหมาย
ติดตามรถเป็น Lot รายสัปดาห์โดยไม่บังคับระบุวันเข้าแน่นอน

## วิธีใช้
1. เข้าเมนู Weekly PM Control และเลือกเดือน
2. ดาวน์โหลด Template Excel หรือใช้ไฟล์เดิมที่มีคอลัมน์ เบอร์รถ/ทะเบียนรถ, Week และกลุ่ม
3. กด Import Lot และเลือกไฟล์
4. เลือกสัปดาห์ 1–5 เพื่อตรวจรายการ
5. เมื่อรถเข้าแล้ว กด ✓ เข้าแล้ว รายการจะไม่ถูกส่งติดตามอีก
6. หากติดเงื่อนไขงานเข้า กด เลื่อน พร้อมใส่เหตุผล
7. กด Copy รายการค้าง หรือ ส่งรายการค้างเข้า LINE

## การเชื่อมกับ My Today
เมื่อกด One Click Complete ใน My Today ระบบจะปิดรายการ Weekly Control ที่มีเบอร์รถตรงกันให้อัตโนมัติ

## สำรองข้อมูล
ข้อมูลใหม่ถูกเก็บใน pm_tracking.db ตาราง weekly_campaign และ weekly_campaign_item ควรสำรองฐานข้อมูลก่อนอัปเดตเวอร์ชัน
