# PM Assistant v1.5 — Sprint 1 & Sprint 2

## Sprint 1: Productivity Core
- PM Assistant Home เป็นหน้าแรก
- Smart Queue เรียงงานตาม Priority
- One Click Complete
- Morning Brief
- Daily Mission Progress

## Sprint 2: Learning & Adoption
- PM Assistant Center
- Interactive Help 4 ขั้นตอน
- PM Handbook ในระบบ
- Workflow Map
- Training Center พร้อมบันทึก Progress ใน Browser

## ฐานข้อมูล
ใช้ไฟล์ `pm_tracking.db` เดิมและไม่ล้างข้อมูล

## เริ่มใช้งาน
1. เปิด `run.bat`
2. หน้าแรกคือ PM Assistant Home
3. กด `เริ่ม Smart Queue`
4. เมื่อเสร็จจริงกด `Complete`
5. เปิด PM Assistant Center เพื่อดูคู่มือหรือเริ่ม Interactive Help
