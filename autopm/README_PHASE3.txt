AutoPM Phase 3: Quick Action + Executive Drilldown

เพิ่มความสามารถในหน้า Summary:
1. กดตัวเลขในตาราง Summary เพื่อแตกดูรถเป็นรายการ
2. เพิ่ม KPI Strip ภายใน Drilldown เช่น ทั้งหมด / เกินระยะ / เรียกเข้า / เตรียม PM / OK / คงเหลือติดลบ / <= 1,000 กม.
3. เพิ่มปุ่ม Quick Action:
   - คัดลอกรายการ: คัดลอกรายการรถแบบพร้อมส่งไลน์/แชท
   - Export CSV: ส่งออกรายการที่กดเป็นไฟล์ CSV เปิดใน Excel ได้
   - เปิดใน Tracking: พาไปหน้าติดตาม PM รายรถพร้อมตั้งฟิลเตอร์ให้อัตโนมัติ
4. กดที่เบอร์รถในรายการ Drilldown เพื่อเปิด Modal รายละเอียดรถทันที

หมายเหตุ:
- ใช้ต่อจาก Phase 2.4 Tracking Filter + M/Y
- อัปโหลด ZIP นี้ขึ้น Netlify ทับ Production Deploy เดิมได้เลย
