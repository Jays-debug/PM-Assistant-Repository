## Interpretation

- FleetOS remains the parent platform.
- AutoPM and PM Assistant remain separate modules.
- Proposed ADR content must not be treated as accepted implementation authorization.
- Acceptance, revision, or supersession requires separate Product Owner approval.
- Repository navigation and formatting repairs do not change ADR status.
